== Description ==

It will add google analytics code on all pages of blogs or websites. FM Google Analytics plugin is lightweight and fast with great features.

**Plugin Features**

* Lightweight and fast performance
* One click setup
* GA Enable/Disable option on the frontend site

[For more information Click Here](https://www.freewebmentor.com/2016/09/google-analytics.html)

Please share your feedback, I would love to hear your feedback, because it will helps to extend the plugin features in future releases.

If you are facing any issues during installation or configuration time, feel free to contact us via [Contact us Page](https://www.freewebmentor.com/contact-us/)

== Installation ==
* Download the archive file and unzip it in /wp-content/plugins, upload the archive at Plugins > Add New > Upload Plugin OR install via Plugins > Add New > Search for the fm google analytics in the search.
* Activate the plugin through the Plugins menu in WordPress
* Navigate to Settings > FM Google Analytics and add web property ID.
* Click Save Settings!
